# E-commerce Data Analysis (SQL + Power BI)

## Overview
This project analyses e-commerce transaction data to identify revenue trends, delivery performance issues, and customer behavior using SQL and Power BI.

## Dataset
- 1M+ records across orders, payments, customers
- Includes order timestamps, delivery status, payment values

## Objectives
- Analyse revenue trends over time
- Identify delayed deliveries
- Detect data inconsistencies
- Understand customer purchasing patterns

## SQL Workflow
- Data cleaning and validation
- Data transformation using joins and aggregations
- Analytical queries for business insights

## Key Insights
- ~10–15% of orders experienced delivery delays
- Identified inconsistencies in payment and order records
- Revenue trends showed strong seasonal patterns

## Dashboard
Power BI dashboard created to track:
- Revenue trends
- Delivery performance
- Order distribution

## Tools Used
- SQL (MySQL)
- Power BI
